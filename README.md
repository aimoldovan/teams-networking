## Install (one time usage)

```sh
git clone https://github.com/aimoldovan/teams-networking.git
cd teams-networking
npm install
```

## Start (Daily usage)

in other terminal open:

```sh
cd ~/Documents/products/node-api
npm start
```

in current project run:

```sh
npm start
```
